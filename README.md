# Ledger AI

Deploy-safe Next.js starter for Ledger AI billing, invoices, products, subscriptions, and payments.

## Routes included

- `/`
- `/login`
- `/dashboard`
- `/customers`
- `/customers/new`
- `/products`
- `/products/import`
- `/invoices`
- `/invoices/new`
- `/subscriptions`
- `/subscriptions/new`
- `/billing`
- `/reporting`
- `/pay/inv_1001`
- `/api/health`

## Run

```bash
npm install
npm run build
npm run typecheck
npm run dev
```
